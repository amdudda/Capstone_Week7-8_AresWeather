Welcome to the Mars Weather Rover site!

This site lets you explore weather data gathered by the Curiosity rover on Mars. The data was downloaded and stored to a database via the Mars Atmospheric Aggregation System API. You can learn more about the {MAAS} API project, or you can read about the Curiosity rover on Wikipedia.

The rover image was lifted from NASA. I believe it qualifies as a "government work" and is therefore not subject to copyright, but if not, let me know at the email link below so I can find a replacement image.

This site was built using Python and Django by A.M. Dudda for the Software Development Capstone course at Minneapolis Community & Technical College.
